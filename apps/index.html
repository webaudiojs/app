Web Network system

Play the light of monet

Interactive Web-audio/visual networking system with node.js in JavaScript. This project aims to find interactive networking collaboration system through web technologies with JavaScript in real-time that to control musical expression and various multimedia factors by Web-based API controller at each different place such as mobile SNS message chat through node.js server

Web-music controller in mobile for Play the light of monet

We have designed four controllers of each different style on the mobile-web such as real instruments. each controller has various features. But generally, all interfaces have similar functions that sound on/off, text chatting, sound/visual processing, volume control, sample sound with buttons, dials, slider, piano keyboard.Main text monitor and on/off switch was set alike.









Web-Audio library

Dr.Sax.js

Dr.Sax.js is Web Audio library for Saxophone in JavaScript. It will help you to create Web Audio applications for music instruments. you can also use in another instruments and voice.
